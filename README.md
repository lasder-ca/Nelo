# Nelo website deployment branch

This branch contains the generated static deployment bundle for the Nelo website.

- Build: `node build-site.mjs`
- Output: `dist/`
- Pages: home, documentation, examples, roadmap, changelog, and 404
- Brand assets: Nelo ownership-boundary mark, favicon, wordmark, and social image

The application source remains on the main development branches. This branch is intentionally isolated so website deployment does not overwrite unpushed framework work.
